## Generic Page Configuration

``` javascript
AWIN.Tracking.DatalyMedia = AWIN.Tracking.DatalyMedia || {};
AWIN.Tracking.DatalyMedia.pagetype = "PAGE_TYPE";
AWIN.Tracking.DatalyMedia.token = "TOKEN";
```
