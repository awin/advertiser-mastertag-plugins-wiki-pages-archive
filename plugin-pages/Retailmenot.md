
# Configuration

Replace `"VALUE_PLACEHOLDERS"` with real values.

``` javascript
AWIN.Tracking.Retailmenot = AWIN.Tracking.Retailmenot || {};
AWIN.Tracking.Retailmenot.mid = "ADVERTISER_ID";
AWIN.Tracking.Retailmenot.basket_amount = "AMOUNT";
```
