# Configuration

Default value for container ID is "0".
Replace `"VALUE_PLACEHOLDERS"` with real values.

## Configuration

``` javascript
AWIN.Tracking.Adality = AWIN.Tracking.Adality || {};
AWIN.Tracking.Adality.containerId = "CONTAINER ID";
```
